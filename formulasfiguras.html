Fórmulas de Figuras


Figuras 3D


Cilíndro



Dados:
r = Raio
h = Altura
Volume = r x h
Área da Base = π x r²
Área Lateral = 2 x π x r x h
Área Total = Área da Base + Área Lateral

Cone



Dados:
r = Raio
h = Altura
g = Geratriz
g = √(r² + h²)
Volume = (π x r² x h)÷3
Área = πr x (r + g)

Cubo



Dados:
a ou l = Lado
Volume = L³
Área = (L²)x6
Perímetro = L x 12




Esferas



Dados:
r = Raio
π = PI
Volume = (3π x r²)÷4
Área = 4π x r²


Paralelepipedos



Dados:
a = Comprimento
b = Espessura
c = Altura
Volume = a x b x c
Área = (a x b) + (a x c) + (b x c)

Pirâmides



Dados:
l = Lado Base
h = Altura
AB = Área da Base (Muda de acordo com a base)
AL = Área Lateral
Volume = (AB x h)÷3
Área = AB + AL


Prismas



Dados:
h = Altura
L = Lado
n = Número de Lados
Volume = AB x h
AL = L x h x n
AT = (2 x AB) + AL


Figuras Planas


Círculo



Dados:
r = Raio
Área = π x r²
Perímetro = 2 x π x r


Losângulo



Dados:
D = Diagonal Maior
d = Diagonal Menor
L = Lado
Área = (D x d)÷2
Perímetro = L x 4

Paralelogramos



Dados:
b = Base
h = Altura
l = Lado Menor
Área = b x h
Perímetro = 2 x (b + l)


Polígonos



Dados:
n = Número de Lados
l = Lado
Área = ((l² x √3)÷4) x n
Perímetro = l x n

Quadrados



Dados:
l = Lado
Áea = l²
Perímetro = l x 4

Retângulos



Dados:
b = Base
h = Altura
Área = b x h
Perímetro = 2 x (b + h)


Trapésios



Dados:
b = Base Menor
B = Base Maior
l = Lado Menor
L = Lado Maior
h = Altura
Área = ((b + B) x h)÷2
Perímetro = b + B + l + L

Triângulos



Dados:
b = Base
h = altura
l = Lado
Área = (b x h)÷2
Perímetro = l1 + l2 + l3
